# Crownfall : The Shattered Kingdom

Crownfall est un jeu de stratégie massivement multijoueur co-localisé, conçu pour être joué dans un amphithéâtre.

## ⚔️ Concept
Le jeu propose une expérience hybride unique :
- **Écran Public (Godot) :** Affiche la carte globale, les frontières et le classement;
- **Écran Privé (Web/Mobile) :** Sert de manette et affiche les ressources secrètes (Or, Puissance).
- **Diplomatie Réelle :** Aucune fonction de chat n'est intégrée pour forcer les interactions physiques et le bluff entre voisins.

## 🛠️ Stack Technique
- **Backend :** Node.js avec Socket.io (WebSockets).
- **Client Mobile :** HTML5 Canvas pour une compatibilité maximale.
- **Affichage Public :** Godot Engine pour les effets visuels.
